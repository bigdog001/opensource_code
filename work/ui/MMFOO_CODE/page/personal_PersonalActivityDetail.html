2011, Jeep70华诞, 在光阴的旅程中在漫长的岁月也不过一瞬，真正重要的不在时间的长久或路途的遥远, 而在于你以怎样的方式, 书写下属于自己的故事。2011, Jeep极致之旅再启征程，三条充满非凡故事的路线, 一群时代英杰。我们与70年SUV缔造者与引领者一起, 回望1942年的烽火, 探寻心中的香格里拉, 更首次穿越国境, 再次书写下自己的非凡人生故事。
